# Standing vs Max Vertical Jump — Key Distinctions Explained

> **Source:** https://www.tiktok.com/@jumpsbychi/video/7564848380271283487
> **Date:** 2026-06-01

CONFIDENCE: medium

## Core Principles

- "Vertical jump" is not a single metric — it simply means jumping for height (jumping vertically)
- Both standing and approach/running jumps are legitimate vertical jumps; neither is "cheating"
- Different vertical jump types measure similar but distinct athletic abilities

## Key Frameworks & Progressions

**Types of Vertical Jumps:**
- **Standing vertical jump** — no approach, measured from a stationary position
- **Max vertical / approach vertical jump** — uses a running or step approach to maximise height
- **Hands-on-hips jump** — removes arm swing as a variable
- **Seated vertical jump** — isolates leg power further

**How Major Leagues Measure:**
- **NFL Combine** — measures standing vertical jump only
- **NBA Draft Combine** — measures *both* standing vertical and max vertical jump

## Things to Avoid

- Dismissing approach/running vertical jumps as illegitimate or "cheating"
- Treating "vertical jump" as a single universal metric without specifying the type
- Comparing standing and max vertical numbers as if they are the same measurement

## Key Takeaways

- There is no single "vertical jump" — always clarify which type is being referenced
- Standing vertical and max vertical are two different metrics, neither superior; they measure related but distinct qualities
- The NBA tracking both measurements signals that each has independent athletic value
- Context matters: when evaluating a jump claim, identify the type before drawing conclusions
